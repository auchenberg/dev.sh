dev

zero config wildcard *.dev domains.

Tired of adding entries to /etc/hosts? Ever wanted wildcard domains in your /etc/hosts?

dev solves all of that by installing a small dns server, and points all *.dev traffic to your computer.

Get started

To install simply open a terminal and run this command:

$ (View Source)

What's going on?

dev is a small shell script that installs dnsmasq, a lightweight DNS server, registeres it as a LaunchDaemon, so it start's when your computer boots, and configures Resolver in OSX to resolve *.dev traffic to dnsmasq.

That's it.
